# FedSched-5G
Este trabalho propõe um algoritmo de escalonamento para tarefas de Aprendizado Federado em redes 5G considerando tráfego de fundo interferente e limitações dos recursos de comunicação. A implementação é integrada ao módulo 5G-LENA do ns-3 e demonstra melhorias na convergência e nos indicadores de desempenho da rede 5G.
