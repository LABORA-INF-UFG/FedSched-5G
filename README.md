# FedSched-5G
Este trabalho propõe um algoritmo de escalonamento para tarefas de Aprendizado Federado em redes 5G considerando tráfego de fundo interferente e limitações dos recursos de comunicação. O algoritmo seleciona dispositivos com dados mais representativos e com melhores condições de canal, emprega uma heurística de agendamento que prioriza os fluxos de atualização de modelo com coexistência de tráfego de fundo e aplica uma estratégia de agregação que reforça as contribuições que favorecem a convergência do modelo global. A implementação é integrada ao módulo 5G-LENA do ns-3 e demonstra melhorias na convergência e nos indicadores de desempenho da rede 5G, superando os algoritmos tradicionais de escalonamento.
